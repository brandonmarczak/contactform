# Web Contact Form

A web contact form using Node.js/Expresss and Nodemailer.js to send emails

*IMPORTANT*
In order for this app to work, please provide your own SMIP info.

## Install Dependencies
```bash
npm install
```
## Run
```bash
node app
```
